# NutritionAR

Augmented Reality iOS application that visualizes product nutritional information in AR. The app also contains a simple game tailored for children to inspire eating healthy. 
Products are detected through barcode scanning and the nutrition data is retrieved from an open-source online database. It is made with SwiftUI, Appleâ€™s Vision framework, ARKit, RealityKit and Reality Composer.
